Simple Page to Get Data from API

Enter a Time Zone in the field below to get the time and date in that Time Zone (leave the field empty for GMT 0)

About: This page uses a static front-end (just Html, CSS and JavaScript) hosted on GitHub Pages,
and a separated OutSystems API where the data is fetched from.
You can try the API directly by clicking the link below:
https://personal-0kioaskk.outsystemscloud.com/ZZtestPage/rest/SimpleTime_API/TimeZone_API?TimeZone=0
